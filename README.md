# Answer Sheet Maker

解答用紙メーカーは、中学校・高校の教員が定期テストや確認テスト用の解答用紙を素早く作成するためのWebアプリケーションです。

## 現在の状態

- Baseline: v0.4.0
- 構成: 単一HTMLファイル
- 実装: Vanilla JavaScript / CSS / HTML
- 配布用ファイル: `answer_sheet_maker.html`
- GitHub Pages用エントリ: `index.html`

## 使い方

`index.html` または `answer_sheet_maker.html` をブラウザで開いて使用します。

## 開発ドキュメント

- 要件定義: `docs/requirements.md`
- スタートガイド: `getting_started_with_claude_code.md`

## 重要な開発ルール

- v0.10までは単一HTMLファイル構成を維持します。
- `docs/requirements.md` 第4章の必須DOM IDと必須クラス名は変更禁止です。
- 機能追加後は第5章の既存機能カタログを確認し、リグレッションを避けます。
