# リボン機能 実装タスクリスト

このリストは、リボン上に表示されているが未実装または機能が不十分なボタンを、順番に実装していくためのものです。

## 現在機能しているもの

- Undo / Redo
- ファイル操作: 新規作成、JSON読込、JSON保存、印刷/PDF
- 大問操作: 追加、削除
- 挿入: テキストボックス、画像、画像URL
- 用紙: 用紙サイズ、向き/段組み、上下余白
- 表示: 中央線、ページ番号、小問配点、大問合計、PDF解答表示
- 文字/段落: 太字、斜体、下線、左右中央揃え、均等割り付け、上下揃え、文字色
- セルクリア

## Phase 1: よく使う編集ボタン

- [ ] 貼り付け
  - クリップボードから選択セル/テキストボックスへ貼り付ける
  - プレーンテキスト優先、必要ならHTML貼り付けも検討

- [ ] コピー
  - 選択中の文字列、セル内容、テキストボックス内容をコピーする

- [ ] 切り取り
  - 選択中の文字列、セル内容、テキストボックス内容を切り取る
  - Undo対象にする

- [ ] 書式のコピー
  - 文字色、太字、下線、揃え、フォントサイズなどをコピー
  - 次に選択した文字/セルへ適用

- [ ] 取り消し線
  - 選択文字へ取り消し線を適用/解除

- [ ] 上付き/下付き
  - 数式入力で使いやすいよう、選択文字へ適用/解除

- [ ] フォントサイズ拡大/縮小
  - 現在選択中の文字サイズを1段階ずつ変更
  - 選択なしの場合は次に入力する文字サイズへ反映

## Phase 2: 色とマーカー

- [ ] 文字色パレットの拡張
  - 黒、解答色ピンクに加えて、赤/青/緑などを追加
  - ピンクだけを解答色として扱う

- [ ] 蛍光ペン
  - 選択文字に背景色を付ける
  - PDF出力時も反映

- [ ] 解答色の表示状態の改善
  - 現在の入力色が黒/解答色のどちらかをリボン上で明確に表示
  - 選択文字の色をリボンに反映

## Phase 3: 挿入系

- [ ] 図形挿入
  - 直線、四角、丸、矢印を追加
  - 移動、サイズ変更、線種、線幅を設定可能にする

- [ ] 表挿入
  - 行数/列数を指定して表を挿入
  - セル結合は後続フェーズで検討

- [ ] テキストボックス挿入ボタンの重複整理
  - ホーム/挿入タブ双方にあるボタンの役割を統一

## Phase 4: 大問/小問設定

- [ ] ホームタブの「設定」ボタン
  - 左ペインの該当大問設定を開く
  - 選択中の大問がなければ先頭大問を選択

- [ ] 問題番号タブ
  - (1), (2), (3)
  - ①, ②, ③
  - ア, イ, ウ
  - などの番号形式を切り替える

- [ ] 線種タブ
  - 大問枠、区切り線、配点欄の線幅/線種を設定

- [ ] 解答欄テンプレート
  - 記述、計算、方眼、罫線などを再導入する場合の設計
  - まずはシンプル四角を維持し、必要なものだけ追加

## Phase 5: ナビゲーション/表示

- [ ] 左ペインの「ヘッダー」タブ
  - タイトル、年組番号氏名欄、日付などをまとめて編集

- [ ] 左ペインの「設定」タブ
  - 用紙、余白、表示オプションをまとめる

- [ ] ヘルプタブ
  - 使い方の簡易表示
  - 外部リンクではなく、アプリ内の最小説明にする

- [ ] クイックアクセスツールバー
  - 保存/Undo/Redo以外のカスタマイズボタンは非表示または実装

## Phase 6: テキストボックス/画像の詳細編集

- [ ] テキストボックスの書式適用範囲を統一
  - セルと同じ文字色、太字、下線、揃えを利用できるようにする

- [ ] 画像の前面/背面
  - テキストボックス、解答欄、画像同士の重なり順を制御

- [ ] 複製
  - 選択中のテキストボックス/画像/図形を複製

## 実装時の注意

- 既存のJSON保存形式を壊さない
- 変更ごとに `index.html` と `answer_sheet_maker.html` を同期する
- 実装ごとにスクリプト構文チェック、`git diff --check`、2ファイル一致確認を行う
- ボタンを実装しない場合は、表示しないか `disabled` にして「押せるのに動かない」状態を避ける
